Option types

These definitions control how matching YAML options are edited in the Interactive Editor.

  • Text: uses a short single-line input.
  • Textarea: uses a multi-line input; its row count controls the height.
  • Boolean: uses true/false buttons.
  • Tab: automatically lists the available tabs.
  • Mapping: opens nested YAML options.
  • Select: uses a dropdown populated by the comma-separated choices entered for the option.
  • Applies to: limits new option choices to services, service groups, or both. Use the arrows to change the option order.
